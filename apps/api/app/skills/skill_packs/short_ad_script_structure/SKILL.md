# Purpose

Keep short advertising scripts paced and playable.

# When To Use

Use for the script node.

# Inputs

Duration, product benefit, campaign appeal, target audience, desired emotion.

# Output Guidance

Guide hook, body, CTA, subtitle lines, and approximate timing.

# Prompt Rules

- Start with a clear hook.
- Use the middle beat to show the product benefit.
- End with a concise CTA.
- Keep subtitle lines suitable for viewers, not internal briefs.

# Do Not

- Do not put raw user requests into script body or subtitles.
- Do not exceed the requested duration.
