# Purpose

Help scene references support multiple storyboard shots.

# When To Use

Use for scene-design and storyboard planning.

# Inputs

Scene specs, storyboard needs, visual style, aspect ratio, channel constraints.

# Output Guidance

Describe scene continuity details and possible camera viewpoints.

# Prompt Rules

- Preserve spatial structure across views.
- Keep color palette and lighting stable.
- Make references useful for image and video prompts.

# Do Not

- Do not create conflicting scene layouts.
- Do not mix unrelated environments in one scene reference.
