# Purpose

Turn storyboard beats into video prompt guidance with timing and motion.

# When To Use

Use for storyboard and storyboard-video-generation preparation.

# Inputs

Storyboard scenes, image references, character turnarounds, scene references, product images.

# Output Guidance

Provide shot prompt, duration, camera motion, action, and input asset guidance.

# Prompt Rules

- Use legal segment durations for the target video model.
- Include reference asset ids per segment.
- Keep camera motion clear and limited.

# Do Not

- Do not submit a full 30 second ad as one short video task.
- Do not use local file paths as remote image URLs.
