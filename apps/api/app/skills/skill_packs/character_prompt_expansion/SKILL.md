name: Character Prompt Expansion
description: Expand character visual prompt notes without adding scenes.
---

# Purpose

Make character descriptions detailed enough for consistent image generation.

# When To Use

Use for the character-design node and character image generation preparation.

# Inputs

Character specs, brand tone, target audience, visual style, character references.

# Output Guidance

Add stable visual details such as face shape, hair, outfit, body type, age, and temperament.

# Prompt Rules

- Keep details consistent across all future shots.
- Separate character identity from action and environment.
- Mention clothing and silhouette when useful.

# Do Not

- Do not add complex scenes.
- Do not change character identity between views.
