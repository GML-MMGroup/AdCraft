# Purpose

Make BGM structure fit the ad duration and emotional arc.

# When To Use

Use for the bgm node.

# Inputs

Duration, script beats, storyboard timing, desired emotion, channels.

# Output Guidance

Provide intro, lift, and ending timing guidance.

# Prompt Rules

- Keep timing aligned to script and storyboard.
- Use smooth transitions.
- Keep loop or fade needs clear.

# Do Not

- Do not ignore the requested duration.
- Do not create distracting music changes.
