# Purpose

Prepare clean scene reference prompts for downstream image generation.

# When To Use

Use for scene-design and scene-image-generation preparation.

# Inputs

Scene specs, product use case, visual style, color and lighting notes.

# Output Guidance

Create pure environment prompt guidance that can become reusable scene images.

# Prompt Rules

- Emphasize architecture, layout, light, color, and mood.
- Keep the scene open enough for later character and product insertion.
- Mention camera angle only when useful for reference consistency.

# Do Not

- Do not include people.
- Do not include unrelated products or props.
