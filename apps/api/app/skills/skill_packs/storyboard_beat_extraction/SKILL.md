# Purpose

Break script and visual design into ordered, executable shots.

# When To Use

Use for the storyboard node.

# Inputs

Script, character design, scene design, character images, scene images, product references.

# Output Guidance

Define scene order, shot type, visual action, text, duration, camera, and input assets.

# Prompt Rules

- Keep each beat suitable for one image and one short video segment.
- Match total duration with requested ad length.
- Use available reference assets when relevant.

# Do Not

- Do not create unfilmable abstract beats.
- Do not ignore product, character, or scene consistency.
