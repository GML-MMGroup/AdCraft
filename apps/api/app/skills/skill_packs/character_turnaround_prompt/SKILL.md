# Purpose

Guide generation of a pure character turnaround sheet for identity consistency.

# When To Use

Use for character-design and character-image-generation preparation.

# Inputs

Character design, visual style, brand constraints, existing character reference images.

# Output Guidance

Describe one character per sheet with front, side, and back views on a plain background.

# Prompt Rules

- Request front view, side view, and back view of the same character.
- Keep face shape, hair, outfit, body type, age, and temperament consistent.
- Use a white, plain, or light gray design sheet background.

# Do Not

- Do not add complex scenes.
- Do not add extra people.
- Do not change clothing or identity across views.
