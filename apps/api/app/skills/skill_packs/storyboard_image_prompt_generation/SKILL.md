# Purpose

Guide storyboard frame prompts using product, character, and scene references.

# When To Use

Use for storyboard and storyboard-image-generation preparation.

# Inputs

Storyboard scenes, product references, character turnarounds, scene references, script.

# Output Guidance

Describe prompt constraints for each storyboard image.

# Prompt Rules

- Preserve product packaging, color, logo, shape, and proportions.
- Preserve character face, hair, outfit, body type, age, and temperament.
- Preserve scene layout, lighting, and color mood.

# Do Not

- Do not mix reference identities.
- Do not omit required product visibility when product showcase is needed.
