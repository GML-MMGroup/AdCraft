# Purpose

Select the most useful audience-facing benefit from product facts and campaign goals.

# When To Use

Use for requirements-analysis and product-design nodes.

# Inputs

Product facts, target audience, campaign goal, desired emotion, channel constraints.

# Output Guidance

Return one primary selling point and optional secondary support points.

# Prompt Rules

- Link each selling point to a target audience need.
- Keep the primary selling point short enough to guide script and visuals.
- Prefer benefits that can be shown visually.

# Do Not

- Do not list every product feature equally.
- Do not use unverifiable superlatives.
