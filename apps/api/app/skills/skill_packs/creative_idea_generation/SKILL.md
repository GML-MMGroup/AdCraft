# Purpose

Create a filmable short ad idea that connects product value to audience motivation.

# When To Use

Use for the script node.

# Inputs

Requirements, product design, creative direction, channel and duration constraints.

# Output Guidance

Provide the central idea that script hook, body, and CTA should express.

# Prompt Rules

- Keep the idea concrete and visual.
- Make the product appear as the solution, not a background prop.
- Fit the idea into the requested duration.

# Do Not

- Do not generate abstract slogans without scene potential.
- Do not include production notes as audience-facing dialogue.
