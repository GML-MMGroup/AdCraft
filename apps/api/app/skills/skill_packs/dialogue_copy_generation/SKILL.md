# Purpose

Ensure script text is suitable for final viewers.

# When To Use

Use for the script node.

# Inputs

Script draft, product benefit, audience tone, channel constraints.

# Output Guidance

Create short spoken or on-screen lines that can become subtitles or narration.

# Prompt Rules

- Use natural, audience-facing language.
- Keep each line short and easy to subtitle.
- Match the desired emotion.

# Do Not

- Do not include user brief text.
- Do not include internal planning labels or workflow instructions.
