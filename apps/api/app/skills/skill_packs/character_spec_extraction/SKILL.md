# Purpose

Define characters that support the ad concept and target audience.

# When To Use

Use for the character-design node.

# Inputs

Requirements, creative direction, script, target audience, selected character assets.

# Output Guidance

Describe each main character with role, appearance, personality, and brand fit.

# Prompt Rules

- Keep character count small for short ads.
- Match appearance and temperament to the campaign tone.
- Preserve identity details for downstream image and video generation.

# Do Not

- Do not add unnecessary extra characters.
- Do not include scene background in pure character design.
