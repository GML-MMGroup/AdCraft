# Purpose

Define environments that support the script and visual style.

# When To Use

Use for the scene-design node.

# Inputs

Requirements, creative direction, script, selected scene references.

# Output Guidance

Describe locations, lighting, color, spatial layout, and atmosphere.

# Prompt Rules

- Keep scene design reusable for multiple shots.
- Make lighting and color specific.
- Separate environment from characters and action.

# Do Not

- Do not add people to pure scene references.
- Do not add extra products unless explicitly required.
