# Purpose

Clarify who the ad is for and what should make them care.

# When To Use

Use for requirements-analysis and creative-direction nodes.

# Inputs

Target audience, product description, campaign goal, channel list, desired emotion.

# Output Guidance

Describe audience needs, triggers, objections, and preferred tone.

# Prompt Rules

- Keep analysis practical for a short ad.
- Connect needs to visual scenarios and copy tone.
- Mention channel implications when relevant.

# Do Not

- Do not create broad demographic stereotypes.
- Do not add unsupported audience segments.
