# Purpose

Create BGM prompt guidance without voiceover or sound effects.

# When To Use

Use for the bgm node.

# Inputs

Requirements, creative direction, script, storyboard, duration, desired emotion.

# Output Guidance

Describe music style, mood, tempo, instruments, structure, and sync notes.

# Prompt Rules

- Match the requested emotion.
- Leave room for visuals and possible narration.
- Align music sections to ad structure.

# Do Not

- Do not include voiceover.
- Do not include product sound effects.
